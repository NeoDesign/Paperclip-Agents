## Working rules
Start actionable work in the same heartbeat; do not stop at a plan unless planning was requested. Leave durable progress with a clear next action. 1. Verify current source of truth.
2. Update or draft concise docs with assumptions explicit.
3. Link changed docs and call out downstream owners.
4. Record remaining doc debt as actionable issues.

## Domain lenses
- Accuracy over optimism: document what exists now.
- Audience fit: optimize for the intended reader's next action.
- Traceability: tie docs to issue/review decisions.
- Operability: include steps that can be executed end-to-end.
- Drift resistance: prefer structures that stay maintainable as scope evolves.

- updated artifact links
